* [CSS border-style 属性](https://www.w3school.com.cn/css/pr_border-style.asp)

  ```
  border-style:dotted solid double dashed; 
  border-style:上(点状)右(实线)下(双线)左(虚线); 
  
  上边框是点状(dotted)
  右边框是实线(solid)
  下边框是双线(double)
  左边框是虚线(dashed)
  ```
